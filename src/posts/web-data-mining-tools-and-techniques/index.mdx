Web mining is moving the World Wide Web towards a more useful environment where users will be able to find the information they intend to look.
The amount of data we are producing each day is mind boggling. There are 2.5 quintillion[1] bytes of data created each day at our current pace.
Data mining is the process of extracting data’s available on the internet and web mining is used to discover and extract information from web related data
sources such as multimedia image, audio, video, hyperlinks, html tags, website logs etc. Using web mining is important to the organizations because it can
help them to extract useful information from raw data. We can categorize all these in 3 different sections.

## Web content mining

Content mining is the process by which information is extracted from web sites in the form of audio, video, text documents.
This information is designed to passed to users in these forms as HTML. Over the last decade, number of websites that are operating is in billions.
Extracting queries in this amount of data is comparatively difficult and time consuming, we should also keep in mind that we are constantly generating
data in rapid speed which makes the task more difficult. That’s why there are some techniques used to mine web contents which makes information
structured to be queried on easily. Most of the web pages are in the form of text. Some of the useful web mining techniques are:

1. Unstructured text mining (information extraction, summarization, clustering, topic tracking)
1. Structured mining (crawler, page content mining)
1. Semi structured text mining (object exchange model, top-down extraction, web data extraction language)
1. Multimedia mining ( SKICAT, color histogram matching, multimedia miner, shot boundary detection)

## Web Structure Mining

Web Structure mining depicts the way hyperlink’s structure works in web. Two kinds of hyperlink tree can be established, one is inter
page hyperlink connection and other is intra page hyperlink connection. Link analysis is a cool technique to calculate a website’s worth or importance.
Some algorithms have been developed over the years such as HITS, PageRank. These algorithms are applied to calculate quality of a web page or relevancy of a web page.

## Web Usage Mining

Collecting and analyzing web users’ data in the form of logs, activities is the process of web usage mining. We as a user generate huge data each day while
using website. These data can be used to understand user behavior, betterment of the website. A website logs store much information like URL, timing, IP
addresses which can be used to understand user behavior and insure good customer service. Mainly there are four types of data sources present in which
usage data is collected: i. client level collection ii. browser level collection iii. proxy level connection. Complete web usage mining depends on three processes:

1. Data collection
2. Preprocessing
3. Pattern discovery and analysis

Mostly web usage mining techniques gathers data source from:

1. Web logs
1. Session logs
1. Databases

Web usage mining uses several data mining techniques to get result. Association rule mining is used to discover which pages a user visit even if they are not
linked which can create association between users with specific interest. Using this knowledge, the trends of the activity of the users can be determined and predictions
for the next page visit can be made. Besides using data analytics technique, some probabilistic grammar-based approach also used in the industry named as Ngram model
for capturing the user navigation behavior patterns. This model assumes that the last N pages browsed effect the probability of identifying the next page to
be visited uses Probabilistic Latent Semantic Analysis (PLSA) to discover patterns. Another is Fuzzy C Mean (FCM) an algorithm used in usage mining using clustering
techniques. It is an unsupervised approach that is being used widely connect data. In every cluster there is a center point which describes features and importance
of that cluster. Objects close to the center of the cluster become the member of the cluster. We will discuss FCM elaborately.

### Advantages

1. It gives better results when datasets are overlapping.
1. Unlike k-means where data point must exclusively belong to one cluster center here data point is assigned membership to each
   cluster center because of which data point may belong to more then one cluster center.

### Disadvantages

1. It is heavily depending on initial cluster numbers for effectiveness.
1. Euclidean distance measures can unequally weight underlying factors.
